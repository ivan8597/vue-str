# Vue Shop

Интернет-магазин на Vue 3 + TypeScript

## Технологии

- Vue 3
- TypeScript
- Pinia
- Vuetify
- Vitest

## Установка
bash
установка зависимостей
npm install
запуск dev сервера
npm run dev
## Функционал

- Просмотр товаров
- Корзина
- Оформление заказа
- История заказов


## Тесты


запуск тестов
npm run test
